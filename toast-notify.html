Toast Notify

Notifikasi mengambang yang ringan dan non-blokir untuk memberikan umpan balik kepada pengguna.

ALERTS
Komponen UI Toasts
Toastify JS (Modern & Lightweight)
Gradient & Basic Styles

Gaya modern menggunakan vanilla JS tanpa dependensi.

JS Code 
Toastify({
  text: "Data berhasil disimpan!",
  duration: 3000,
  gravity: "top", // top or bottom
  position: "right", // left, center or right
  style: {
    background: "linear-gradient(135deg, #10b981, #3b82f6)",
  }
}).showToast();
Positioning

Posisi fleksibel di layar.

JS Code 
Toastify({
  text: "Notifikasi di kiri atas",
  gravity: "top", 
  position: "left",
  style: { background: "#3b82f6" }
}).showToast();
SweetAlert2 (Toast Mode)
SweetAlert Mixin

Menggunakan SweetAlert2 sebagai toast notification yang elegan.

JS Configuration 
const Toast = Swal.mixin({
  toast: true,
  position: 'top-end',
  showConfirmButton: false,
  timer: 3000,
  timerProgressBar: true,
  background: '#1f2937', // Sesuaikan tema
  color: '#fff'
});

Toast.fire({
  icon: 'success',
  title: 'Signed in successfully'
})
Toastr (jQuery Required)
Info & Success
toastr.success('Pesan', 'Judul');
toastr.info('Pesan', 'Judul');
Warning & Error
toastr.warning('Pesan', 'Judul');
toastr.error('Pesan', 'Judul');
Configuration

Contoh pengaturan global untuk Toastr.

toastr.options = {
  "closeButton": true,
  "progressBar": true,
  "positionClass": "toast-top-right",
  "timeOut": "5000"
}
